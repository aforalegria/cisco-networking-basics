# Wireless and mobile networks

- **Cellular provider network**. Divide geographic areas into “cells,” each served by a base station that wirelessly connects nearby devices and hands them off between cells as they move.
These base stations link into a core network that manages routing, authentication, and data services.
- **Wireless Fidelity (WiFi)**. Wi-Fi uses radio waves to wirelessly connect devices to a nearby router, which broadcasts a network signal that devices can join using a password or authentication. 
- **Bluetooth**. short-range wireless technology, uses low-power radio waves to connect nearby devices directly, exchanging data without internet.
- **Global Positioninig System (GPS)**. Uses signals from multiple satellites orbiting Earth. Device measures how long those signals take to arrive and uses that timing to calculate its exact location through triangulation.
- **Near Field Communication (NFC)**. Uses satellites to transmit signals that cover the globe. The smartphone can receive these signals and calculate the phone’s location to an accuracy of within 10 meters.
